Framework · universal method

Four steps.

Most valuation methodologies boil down to four steps: identify value drivers, forecast the future, discount to present value, and divide the pie.

01 · Identify
Value drivers: top-down (TAM, macro) and bottom-up (leadership, brand, unit economics). GIGO applies.
02 · Forecast
Project sales/margins via scenario analysis: base, bull, bear. Private data is scarce.
03 · Discount
PV = FV / (1+r)^n. $100 at 10% for 7 yrs = $51.32 today.
04 · Divide
EV = Net Debt + Equity. Creditors first; shareholders residual. Use as-converted for SAFEs/notes.
Enterprise Value = (Total Debt − Cash) + Equity, where Equity = Preferred + Common + NCI. Going concern always > liquidation.
