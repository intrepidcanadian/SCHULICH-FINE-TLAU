Bitcoin’s open-source progeny

17,000+ coins. 41% Bitcoin.

Satoshi posted the source code on GitHub in 2009. Programmers cloned and modified it: Litecoin, Namecoin, Peercoin, Dogecoin, XRP. As of April 2023, 23,000+ coins. The top 10 = 79% of market cap.

Litecoin (2011)
Scrypt hash — memory-bound, costlier to mine.
Namecoin (2011)
Censorship-resistant DNS, bypasses ICANN.
Peercoin (2012)
First to combine PoW with PoS.
Dogecoin (2013)
Joke coin with the Doge meme — reached new audience.
Hard fork: divorce among holders. Aug 2017 → Bitcoin Cash (8 MB blocks). Oct 2017 → Bitcoin Gold. Nov 2018 → Bitcoin SV. Holders end up with coins on both chains — the sum-of-parts exceeds the original price.
