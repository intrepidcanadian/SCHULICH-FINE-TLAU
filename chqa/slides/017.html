Chapter 6 · Ethereum & DeFi

What is a smart contract? What is a decentralized application (dapp)? How are smart contracts and dapps related to LEGO?

A smart contract is computer code that executes a transaction without human intervention, while a dapp is a web application built on decentralized P2P infrastructure.
—Smart contracts are encrypted on the blockchain to prevent alterations, removing financial intermediaries.
—Smart contracts are the LEGO pieces; dapps and DAOs are what you build with them.
—Smart contracts can be written in Solidity, a Turing-complete language running on the EVM.
—Many dapps combine a smart contract with a front-end user interface.
